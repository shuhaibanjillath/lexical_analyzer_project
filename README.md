# Lexical Analyzer Project in C

This project is a simple Lexical Analyzer written in C. It reads a C source file and identifies different tokens such as keywords, identifiers, constants, operators, and symbols.

## Features
- Detects keywords and identifiers
- Validates numeric constants (decimal, octal, hex, binary, float)
- Recognizes operators and symbols
- Handles string and character literals
- Skips comments
- Reports lexical errors with line numbers
